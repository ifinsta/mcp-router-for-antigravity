## 🎯 What Happens When You Run It

The application will launch two components:

1. **MCP Router Server** (Background)
   - Starts the main MCP server on stdio
   - Handles tool requests from AI agents
   - Manages browser automation

2. **Browser Control Interface** (Main Window)
   - Electron-based GUI application
   - Visual browser control panel
   - Real-time status monitoring

## 🛠️ Features Available

- **Multi-Browser Support**: Chrome, Edge, Firefox, Safari
- **Headless Mode**: Automated browsing without visual interface
- **Network Simulation**: Test different network conditions
- **Device Emulation**: Mobile, tablet, desktop profiles
- **Multi-Tab Management**: Control multiple browser tabs
- **Performance Monitoring**: Track resource usage and response times

## 🔧 Troubleshooting

### Application Won't Start
1. Check if Node.js is installed (v20.10.0+ required)
2. Verify all files are in the installation directory
3. Run as Administrator if permissions issues occur

### Browser Control Not Working
1. Ensure browsers are installed on your system
2. Check browser driver compatibility
3. Verify browser automation permissions

### MCP Server Issues
1. Check console output for error messages
2. Verify port availability (default: stdio transport)
3. Review MCP configuration in `package.json`
